# java-dsa-practice

This repository is a practice workspace for Java and common Data Structures & Algorithms (DSA).

Structure:

- Java/
  - OOP/           # object-oriented programming examples and exercises
  - Collections/   # usage and practice with Java Collections framework
  - Exceptions/    # examples of exception handling
  - JDBC/          # simple JDBC examples and database connectivity

- DSA/
  - Arrays/        # array problems and solutions
  - Strings/       # string algorithms and exercises
  - LinkedList/    # linked list problems
  - Stack/         # stack implementations and problems
  - Queue/         # queue implementations and problems
